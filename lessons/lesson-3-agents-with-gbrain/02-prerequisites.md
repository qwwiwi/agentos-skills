# 02. Prerequisites

Прежде чем начать – проверь, что у тебя установлено и заведено всё нужное. Не пропускай, на середине урока возвращаться за компонентами больно.

## Локально (твой ноутбук)

### Инструменты

| Инструмент | Минимальная версия | Зачем |
|---|---|---|
| Claude Code | последняя | главный исполнитель |
| gh CLI | 2.40+ | работа с GitHub из терминала |
| git | 2.30+ | клонирование репо |
| Node.js | 22+ | install-скрипты, MCP-инспектор |
| Python | 3.12+ | gbrain клиенты, smoke-скрипты |
| curl | любая | smoke-тесты HTTP |
| jq | 1.6+ | парсинг JSON-ответов |

Проверка:

```bash
claude --version
gh --version
git --version
node --version
python3 --version
curl --version
jq --version
```

Если чего-то нет – ставь через системный менеджер (`brew install ...` на Mac, `apt install ...` на Ubuntu).

### Авторизация

- `gh auth login` – авторизуйся в GitHub (нужно для клонирования + создания репо)
- `claude` – запусти, авторизуйся в своём аккаунте

## Аккаунты

| Сервис | Зачем | План |
|---|---|---|
| GitHub | хранение твоих workspace, доступ к публичным репо | бесплатный |
| Anthropic | сам Claude Code | Max ($100–200/мес) или Pro+API |
| Telegram | боты для агентов (по одному на агента) | бесплатный |

**Telegram-боты:** напиши `@BotFather`, создай три бота:

- `your_coder_bot` – для coder-агента
- `your_marketer_bot` – для marketer-агента
- `your_sales_bot` – для sales-агента

Сохрани три токена в безопасное место (понадобятся на шагах 04-05). Боты можно подключить позже, но удобнее завести сейчас.

## Сервер для gbrain

### Минимальные требования

- 4 vCPU
- 8 GiB RAM
- 20 GB SSD
- Ubuntu 22.04 или 24.04 LTS
- Публичный IPv4

### Рекомендуемый провайдер

**Hetzner CPX42** – ~12€/мес, конфигурация ровно подходит (4 vCPU AMD, 8 GiB, 160 GB SSD). Заводи через `console.hetzner.com`, регион выбирай ближе к себе (Nuremberg / Helsinki / Falkenstein).

Альтернативы: DigitalOcean Premium, Vultr High Frequency, Timeweb Cloud, Selectel. Любой VPS с указанными ресурсами подойдёт.

**Не используй**: shared hosting, бесплатные tier'ы (AWS Free Tier у gbrain не хватит RAM), Raspberry Pi (медленные эмбеддинги).

### Доступ

- SSH-ключ должен быть положен на сервер ещё при создании (большинство панелей это делает автоматически)
- Проверка: `ssh root@your-server-ip` – должно зайти без пароля

## Домен

Нужен один домен (или поддомен), куда привяжем gbrain. Например:

- `your-gbrain.example.com` – если у тебя уже есть `example.com`
- купи новый: `.xyz`/`.ru`/`.com` через любой регистратор (Porkbun, Namecheap, REG.RU)

Должна быть возможность прописать A-запись (или AAAA для IPv6). Cloudflare как DNS-провайдер удобен, но для MCP – обязательно ставь A-запись без проксирования (grey cloud в Cloudflare). MCP стримит SSE, проксирование ломает поток.

## Дисковое пространство

- Локально: ~5 GB под три workspace и Claude Code
- VPS: ~20 GB под gbrain (Postgres + vault + логи)

## Чек-лист перед стартом

Проверь по списку – все галки должны стоять. Если нет – вернись и закрой пробел.

- [ ] `claude --version` показывает версию (Claude Code установлен)
- [ ] `gh auth status` показывает «Logged in» (GitHub CLI авторизован)
- [ ] `git config user.name` и `user.email` заполнены
- [ ] `node --version` показывает 22.x или выше
- [ ] `python3 --version` показывает 3.12.x или выше
- [ ] `curl` и `jq` доступны
- [ ] У тебя есть три Telegram-бота с токенами (или ты готов сделать их по ходу)
- [ ] У тебя есть VPS с указанной конфигурацией и SSH-доступом
- [ ] У тебя есть домен (или поддомен), куда привяжем gbrain
- [ ] Готов потратить ~90 минут без перерывов на отвлечения

Всё готово – переходи к [03-create-coder.md](./03-create-coder.md).
